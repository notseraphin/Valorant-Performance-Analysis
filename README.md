Valorant Performance Analysis & Predictive Modeling

This personal project utilizes my games and analyzes Valorant performance using the HenrikDev API, generating insights on:

-Headshot % trends

-KD ratio patterns

-Total engagements vs performance

-Regression and predictive win probability modeling

-Required KD for 50%/60%/70% win probability

-Full Jupyter notebook interactive analysis

Tech Stack:

-Python

-Pandas / NumPy

-Matplotlib / Seaborn

-scikit-learn

-Jupyter Notebook

-HenrikDev Valorant API
